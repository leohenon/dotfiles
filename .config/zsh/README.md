# zsh

Opinionated zsh config. Matches [fish](../fish/README.md).

![zsh](assets/prompt.png)

- vi mode with cursor shape switching
- tab accepts inline suggestion
- shift-tab triggers completion
- shared abbreviations from `../shell/abbrs.tsv`
- `fe` / `fv` / `fp` / `fs` / `ff` / `gstage` / `gunstage` helpers
